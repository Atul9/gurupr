# GURU-PR

Site do Grupo de Usuários Ruby do Paraná!

Utilizamos:
- Github para autenticação
- Heroku
- PostgreSQL

De resto, vamos desenvolvendo aos poucos!

## Instalação

```bash
cp .env.example .env # copie e configure as chaves
bundle install
rake db:create db:migrate
rails server
```
